一、实验目的#

  1. 掌握卷积神经网络、循环神经网络等深度学习的各项基本技术。
  2. 加强对pytorch、tensorflow等深度学习框架的使用能力。

二、实验要求#

  1. 任选一个深度学习框架，实现在给定数据集上的基于DNN、CNN、RNN的图片分类模型。

三、网络结构#

1. DNN#

先将三维图像张量展平为一维向量，然后使用三层全连接网络，激活函数为leakyrelu，隐藏层使用dropout，hidden_dim设置为200，因为是二分类，所以out_dim为2。

代码：

class MLP(nn.Module):
    def __init__(self, in_dim, hidden_dim, out_dim):
        super().__init__()
        self.dropout = nn.Dropout(0.2)
        self.flatten = nn.Flatten()
        self.linear_relu_stack = nn.Sequential(
            nn.Linear(in_dim, hidden_dim),
            nn.LeakyReLU(),
            self.dropout,
            nn.Linear(hidden_dim, hidden_dim),
            nn.LeakyReLU(),
            self.dropout,
            nn.Linear(hidden_dim, out_dim),
        )

    def forward(self, x):
        x = self.flatten(x)
        logits = self.linear_relu_stack(x)
        return logits

2. CNN#

本次实验使用普通的cnn模型，包括三个卷积-池化层和三个全连接网络。因为每次卷积下采样后特征图尺寸减为原来一半，所以要求输入图像尺寸为16的倍数。此外在每次卷积后使用了dropout缓解过拟合。

代码：

class CNN(nn.Module):
    def __init__(self, in_size):
        super(CNN, self).__init__()
        self.dropout = nn.Dropout(0.2)
        self.conv1 = nn.Conv2d(in_channels=3, out_channels=8, kernel_size=3, padding=1)
        self.conv2 = nn.Conv2d(in_channels=8, out_channels=16, kernel_size=3, padding=1)
        self.conv3 = nn.Conv2d(in_channels=16, out_channels=16, kernel_size=3, padding=1)
        self.conv4 = nn.Conv2d(in_channels=16, out_channels=32, kernel_size=3, padding=1)
        out_size = in_size // 16
        self.fc1 = nn.Linear(out_size * out_size * 32, 128)
        self.fc2 = nn.Linear(128, 2)

    def forward(self, x):
        x = F.leaky_relu(self.conv1(x))
        x = F.max_pool2d(x, 2)
        x = self.dropout(x)
        x = F.leaky_relu(self.conv2(x))
        x = F.max_pool2d(x, 2)
        x = self.dropout(x)
        x = F.leaky_relu(self.conv3(x))
        x = F.max_pool2d(x, 2)
        x = self.dropout(x)
        x = F.leaky_relu(self.conv4(x))
        x = F.max_pool2d(x, 2)
        x = self.dropout(x)

        x = x.view(x.size(0), -1)
        x = F.leaky_relu(self.fc1(x))
        x = self.fc2(x)
        return x

3. RNN#

将每一个图像分别以宽度和高度维度展开成序列，对于展开后的序列，使用两层带dropout的双向RNN编码，然后分别对两个方向最终隐藏层输出的特征向量做均值均值池化，然后接一层全连接，得到h_wide与h_height横向与纵向编码。

最后再对两个方向编码做均值池化并输入两层全连接网络求出猫与狗的类别概率。

代码：

class RNN(nn.Module):
    def __init__(self, in_dim, h_dim):
        super().__init__()

        self.rnn = nn.RNN(in_dim, h_dim, num_layers=2, bidirectional=True, dropout=0.2)
        self.fc1 = nn.Linear(h_dim, h_dim)

        self.fc2 = nn.Linear(h_dim, 100)
        self.fc3 = nn.Linear(100, 2)

    def forward(self, x):
        x_h = x.permute(2, 0, 1, 3)
        x_h = x_h.reshape(x_h.shape[0], x_h.shape[1], -1)
        x_w = x.permute(3, 0, 1, 2)
        x_w = x_w.reshape(x_w.shape[0], x_w.shape[1], -1)
        x_h = self.compute_rnn(x_h)
        x_w = self.compute_rnn(x_w)
        x = torch.stack([x_h, x_w], dim=0).mean(dim=0)

        x = F.leaky_relu(self.fc2(x))
        x = self.fc3(x)
        return x

    def compute_rnn(self, x):
        _, h_n = self.rnn(x)  # (2*layer, size(batch), h_dim)
        h_n = h_n[-2:]  # (2, size(batch), h_dim)
        h_n = torch.mean(h_n, 0)  # (size(batch), h_dim)

        h_n = self.fc1(h_n)
        h_n = F.leaky_relu(h_n)
        return h_n

四、实验结果#

1. 实验设置#

使用给定的2000张训练图像、500张验证集图像进行实验。预处理数据预处理仅包括图像尺寸缩放（arg.size设置为128*128）和图像标准化（使用Imagenet均值与方差）。

data_transform = transforms.Compose([
    transforms.Resize([args.size, args.size]),
    transforms.ToTensor(),
    transforms.Normalize(  # standardized processing
        mean=[0.485, 0.456, 0.406],
        std=[0.229, 0.224, 0.225]
    )
])

训练使用Adam优化器，学习率1e-3，权重缩减系数1e-4。batchsize设为64，epoch为20。

2. 结果#

  • 训练损失、准确率：

    DNN模型在训练集上拟合最好，其次是CNN与RNN。

    DNN CNN RNN

  • 验证损失、准确率：

    在验证集上，DNN在第5个epoch获得最好准确率64.00%；CNN在第17个epoch取得最好准确率75.20%；RNN在第14个epoch取得最好准确率59.40%。

    DNN CNN RNN

五、问题与思考#

  • 问题

    为了保证实验可复现性，要设置一些随机种子并禁用随机算法。对于包含nn.RNN的模型，pytorch官方文档给出需要额外设置：

    There are known non-determinism issues for RNN functions on some versions of cuDNN and CUDA. You can enforce deterministic behavior by setting the following environment variables:

    On CUDA 10.1, set environment variable CUDA_LAUNCH_BLOCKING=1. This may affect performance.

    On CUDA 10.2 or later, set environment variable (note the leading colon symbol) CUBLAS_WORKSPACE_CONFIG=:16:8 or CUBLAS_WORKSPACE_CONFIG=:4096:2.

    我运行环境的CUDA版本是11.3，所以设置了CUBLAS_WORKSPACE_CONFIG=:4096:2。但是在实验中RNN模型仍然是不可复现的。

    最终在pytorch的github仓库的issue发现LSTM也有类似的问题，并且这是一个cuDNN的一个bug。经我测试对于nn.RNN模块与nn.LSTM问题相同，即在同时满足下面条件的情况：①layer$\geq$2；②bidirectional为True；③dropout设置大于0；结果就会随机。

    此外，该问题下有个评论与我运行环境相同，也证明LSTM有这样的问题，并指出似乎在torch1.12 rc版本LSTM不存在这个问题。

  • 思考

    1. 使用dropout可以在一定程度上略微缓解过拟合问题。
    2. 从上一部分的训练结果中可以看出，DNN和RNN在训练集上损失较小，准确率较高，但是在验证集却损失很大、准确率较低。DNN由于其模型参数较大导致过拟合；而RNN不是为图像识别任务所设计的模型，导致其在训练集上的效果就不太好，所以在验证及上的泛化效果也较差。
    3. CNN是唯一在20epoch内没有过拟合的模型（验证损失持续下降），并且其最终取得了75.20%最好的准确率，可见CNN的滤波器的局部连接和权值共享特性能够很好地提取图像特征从而适用于图像分类任务。
